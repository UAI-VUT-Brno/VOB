# SOLID - Worker priklad

Vsechny ukazky jsou pouze na domene `Worker`.

1. `01_bad_all_solid` - zamerne spatny kod, porusuje SRP, OCP, LSP, ISP i DIP.
2. `02_fix_srp` - oprava SRP.
3. `03_fix_ocp` - oprava OCP.
4. `04_fix_lsp` - oprava LSP.
5. `05_fix_isp` - oprava ISP.
6. `06_fix_dip` - oprava DIP.

Kazdy dalsi krok navazuje na predchozi a opravuje prave jeden princip.
